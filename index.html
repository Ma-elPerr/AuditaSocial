Sobre a Ferramenta

AuditaSocial é uma solução inovadora que permite a identificação de possíveis fraudes através do cruzamento de dados entre servidores públicos e beneficiários de programas sociais.

Auditoria Automatizada

Cruzamento automático de dados entre folhas de pagamento e cadastros de beneficiários.

Conformidade Legal

Baseado na Lei de Acesso à Informação e alinhado com as melhores práticas de integridade pública.

Inteligência de Dados

Análise preditiva para identificar padrões de irregularidades com alta precisão.

Casos de Uso

Exemplos práticos de como o AuditaSocial pode ser aplicado para combater fraudes e promover a integridade.

Caso Real: Médico Municipal

Um município utilizou o AuditaSocial para identificar um médico municipal que recebia o benefício do Bolsa Família, apesar de ter renda superior ao permitido pelo programa.

A ferramenta cruzou automaticamente os dados dos servidores municipais com o cadastro de beneficiários federais, identificando a irregularidade em segundos.

  • Processo Administrativo Disciplinar (PAD) instaurado
  • Cancelamento do benefício irregular
  • Benefícios para os cofres públicos
  • Recurso realocado para famílias necessitadas

Guia de Instalação e Uso

Siga os passos abaixo para configurar e executar a ferramenta AuditaSocial em sua máquina.

1. Instalar o Docker Desktop

  1. Acesse o site oficial e baixe o instalador: Docker Desktop.
  2. Clique em Download Docker Desktop para o seu sistema operacional (ex: Windows).
  3. Após o download, execute o instalador "Docker Desktop Installer" com as configurações recomendadas. Reinicie o computador se for solicitado.
  4. Crie uma conta Docker ou faça login usando sua conta Google/GitHub.
  5. Abra o Docker Desktop, vá em Settings (ícone de engrenagem) > Resources > Network e habilite a opção "Enable host Network". Salve as alterações.
  6. Deixe o Docker Desktop rodando em segundo plano.

2. Baixar a Ferramenta AuditaSocial

  1. Acesse o repositório: AuditaSocial no GitHub.
  2. Clique no botão verde "<> Code" e selecione "Download ZIP".
  3. Extraia o conteúdo do arquivo ZIP para uma pasta de sua preferência.

3. Rodando o Código

  1. Abra a pasta onde você extraiu os arquivos. Clique com o botão direito do mouse dentro da pasta e selecione "Abrir no Terminal" (ou "Open in Terminal").
  2. No terminal que se abriu, digite o comando abaixo e pressione Enter. (Você receberá uma mensagem de erro se o Docker não estiver aberto).
docker compose up
  1. Aguarde o processo terminar. A confirmação aparecerá no terminal quando estiver pronto.
  2. Volte para a pasta do sistema e dê um duplo clique no atalho "Iniciar Sistema AuditaSocial".

4. Usando o Sistema AuditaSocial

  1. Clique no botão para Atualização de Bases Sociais e aguarde o preenchimento dos dados.

    2. Aguarde até que todas as bases sociais sejam preenchidas na planilha abaixo.

  2. Selecione o seu arquivo contendo as informações dos servidores a serem consultados. (Caso necessário, faça o download de um documento modelo).
  3. Importante: sua planilha deve conter as mesmas cinco colunas que as planilhas modelo.
  4. Clique em Executar o Cruzamento.
  5. Caso queira, clique em Fazer o download do resultado para obter uma planilha com os casos positivos identificados.
